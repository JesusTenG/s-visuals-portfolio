# Hero background loops

Place short, muted, optimized MP4 loops here (recommended: 5–15s, ≤720p, no audio, &lt;2MB each).

Example:

- `clip-01.mp4`
- `clip-02.mp4`

Register each file in `src/components/hero/heroVideos.ts`:

```ts
{ id: "clip-01", src: "/assets/videos/hero/clip-01.mp4" },
```

Use preview encodes only as a temporary stand-in — never lightbox sources.

No poster images are used in the Hero panel experiment.
